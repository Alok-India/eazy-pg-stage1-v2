# Eazy-PG Stage 1

Lead-generation website for **Eazy-PG** in **Greater Noida West, India**.

## Included in this Stage 1 MVP

- Landing page / hero section
- Room options section
- Amenities section
- Gallery placeholder section
- FAQ / trust section
- Contact enquiry form UI
- WhatsApp call-to-action buttons

## Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`

## Hosting mode

This project is now configured for **static export** so it can be uploaded to **Hostinger shared hosting**.

## Build for Hostinger

```bash
npm install
npm run build
```

After build, upload the contents of the `out/` folder to your Hostinger `public_html` directory.
